## 架构变迁

| 维度 | v1（Unity） | v2（当前） |
|------|-----------|----------|
| 大屏展示 | Unity 2022.3 + DOTween | Vue 3 + CSS 动画 |
| 人脸采集 | Unity WebCamTexture + ArcFace DLL | Browser getUserMedia → 后端 JNA 调用 DLL |
| 特征提取 | Unity C# DllImport | Spring Boot JNA 桥接 |
| 大屏推送 | WebSocket → Unity | 浏览器轮询 + WebSocket 可选 |
| 部署方式 | Unity 独立 .exe | 浏览器全屏 |

## 迭代方法论（Vibe Coding 实践）

1. **需求拆解**：将大需求拆分为可独立验证的小迭代，每次迭代产出可运行成果
2. **快速反馈**：每完成一个模块立即联调验证，问题早发现早修复
3. **AI 赋能**：使用 Claude Code 辅助编码、代码审查、性能优化、安全审计
4. **持续集成**：Git 规范化提交（feat/fix/perf/chore/security），分支管理清晰
